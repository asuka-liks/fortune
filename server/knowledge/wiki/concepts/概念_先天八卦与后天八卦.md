# 先天八卦与后天八卦

## 定义

易经有两套八卦排列系统，服务于不同的目的：

- **先天八卦（伏羲八卦）**：描述宇宙的**静态本体**——事物在"尚未进入时间"时的本质关系。核心法则：**对峙**。
- **后天八卦（文王八卦）**：描述万物的**动态运行**——事物在"进入时间"后的变化规律。核心法则：**流行**。

## 两种"坐标系"对比

| | 先天八卦 | 后天八卦 |
|------|------|------|
| **别称** | 伏羲八卦 | 文王八卦 |
| **数字来源** | 则河图 | 则洛书 |
| **乾坤位置** | 乾南坤北 | 乾西北，坤西南 |
| **主轴** | 乾坤纵轴 | 坎离子午轴 |
| **运转逻辑** | "天地定位，山泽通气，雷风相薄，水火不相射" | "帝出乎震"——从东方春生开始顺时针循环 |
| **数序** | 乾1兑2离3震4巽5坎6艮7坤8 | 坎1坤2震3巽4 乾6兑7艮8离9 |
| **用途** | 理论推导、梅花易数起卦 | 方位应用、风水、中医 |

### 先天八卦的"对峙"逻辑

```
      乾（南/天）
  兑        巽
离            坎
  震        艮
      坤（北/地）
```

每对对角的卦互为"错卦"（阴阳全反）：乾坤对立、坎离对立、震巽对立、艮兑对立。

### 后天八卦的"流行"逻辑

```
      离（南/夏/火）
  巽            坤
震                兑
  艮            乾
      坎（北/冬/水）
```

顺时针循环：震（春生）→巽（齐长）→离（夏长）→坤（致养）→兑（秋收）→乾（交战）→坎（冬藏）→艮（终成复始）

## 在符号可迁移性框架中的位置

先天/后天之分揭示了符号系统的一个深层设计原则：

| 设计原则 | 说明 | 在Fortune中的应用 |
|----------|------|------------------|
| **双套坐标系** | 同一套符号，两套排列，服务于"理论"和"应用"两个层面 | Fortune可以设计"理论面"（符号的纯粹含义）和"应用面"（在具体语境中的解读变体） |
| **对峙 vs 流行** | 结构 vs 过程——一个符号系统需要同时处理"是什么"和"怎么变" | 卡牌占卜的"正逆位"（对峙）+ 牌阵位置的时间流（流行） |

> 从 [[concepts/卡牌占卜符号映射]] 的视角：先天八卦是一个"干净的"符号映射层（乾=天=纯阳），后天八卦是一个"语境化的"映射变体（乾=西北=季秋=父亲）。二者不是矛盾——它们是同一个符号系统的两个"视角"。这为理解塔罗的"同一张牌在不同牌阵位置含义不同"提供了结构性类比。
