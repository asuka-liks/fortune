## 定义

铜钱摇卦算法是 Fortune 八卦五行模式的核心——用 JavaScript 模拟三枚铜钱抛掷六次，通过二进制编码映射到六十四卦，自动计算动爻、变卦，并输出完整的卦象信息。是易经符号系统在 web 应用中的**可运行实现**。

## 算法全流程

```
三枚硬币 × 6 次抛掷
  → 每次求和 (6/7/8/9) → 老阴/少阳/少阴/老阳
  → 6 个阴阳布尔值 → 上下卦 → 查 King Wen 编号
  → 动爻翻转 → 变卦（之卦）
  → 格式化输出 → AI prompt
```

## 核心数据结构

### 铜钱抛掷

```
三枚铜钱：正面(阳)=3，背面(阴)=2
求和：
  6 (2+2+2) = 老阴 → 是阴爻 + 动爻
  7 (3+2+2) = 少阳 → 是阳爻 + 静爻
  8 (3+3+2) = 少阴 → 是阴爻 + 静爻
  9 (3+3+3) = 老阳 → 是阳爻 + 动爻
```

老阴(6) → 阴爻变阳爻；老阳(9) → 阳爻变阴爻

### 八卦编码

Fortune 的 `getTrigramName()` 使用巧妙的位运算：
```
乾 ☰ = 111 → 7    兑 ☱ = 110 → 3
离 ☲ = 101 → 5    震 ☳ = 100 → 1
巽 ☴ = 011 → 6    坎 ☵ = 010 → 2
艮 ☶ = 001 → 4    坤 ☷ = 000 → 0
```

### 六十四卦查找

通过 `"下卦名_上卦名"` 键值对快速查表，King Wen 编号 1-64，每卦包含：
- 中英文命名 / 上下卦 / 判词 / 卦象描述

## 辅助功能

### 五行生克
`ELEMENT_GENERATES` + `ELEMENT_CONTROLS` 两个映射表定义了五行之间的推演关系，供 prompt 格式化时自动计算体用生克。

### 六亲
`SIX_RELATIONS` 定义父母/兄弟/妻财/官鬼/子孙，在 prompt 中根据日干和卦爻的五行关系自动推算。

### 梅花易数数字起卦
`deriveByPoints(a, b)` 实现了用两张卡牌点数起卦的完整算法：

```
第一数 a ÷8 → 上卦
第二数 b ÷8 → 下卦
(a+b) ÷6 → 动爻
+ 自动计算互卦（2/3/4爻→下互，3/4/5爻→上互）
```

参见 [[entities/三国杀占卜]] 和 Fortune 趣味模式中如何使用这一算法。

## 格式化输出

`formatShakeResult()` 生成 AI 可读的完整卦象文本，包含：
- 六次铜钱抛掷过程和结果
- 本卦名称 + 上下卦 + 五行 + 卦辞
- 变卦信息（如果有动爻）
- 动爻数量和位置

## 在建构论中的意义

这个模块是**第2层（解释规则层）的纯算法实现**：
- 八卦的 2^3=8 种三爻组合是数学事实
- 64 卦的 2^6 全排列也是数学事实
- 铜钱求和 → 老/少/阴/阳 的概率分布是数学事实

这些规则不依赖任何"灵验"假设——它们是纯符号运算。参见 [[concepts/概念_六十四卦]] 和 [[concepts/概念_先天八卦与后天八卦]]。
