## 定义

八字命理（又称四柱推命/子平八字）是中国最主流的命理学体系。将一个人的出生年、月、日、时分别对应天干地支，形成"四柱"（年柱、月柱、日柱、时柱），每柱有一干一支，共八个字，故称"八字"。

## 核心概念

### 天干地支

| 十天干 | 五行 | 阴阳 |
|--------|------|------|
| 甲、乙 | 木 | 甲阳乙阴 |
| 丙、丁 | 火 | 丙阳丁阴 |
| 戊、己 | 土 | 戊阳己阴 |
| 庚、辛 | 金 | 庚阳辛阴 |
| 壬、癸 | 水 | 壬阳癸阴 |

| 十二地支 | 五行 | 生肖 |
|----------|------|------|
| 子、亥 | 水 | 鼠、猪 |
| 寅、卯 | 木 | 虎、兔 |
| 巳、午 | 火 | 蛇、马 |
| 申、酉 | 金 | 猴、鸡 |
| 辰戌丑未 | 土 | 龙狗牛羊 |

### 四柱排法

以公历出生日期 + 出生时辰为基础，通过万年历换算：

- **年柱**：以立春为界，立春前属上一年
- **月柱**：以节气为界，使用"五虎遁"口诀推算月干
- **日柱**：查万年历获取日干支
- **时柱**：日干 + 时辰，使用"五鼠遁"口诀推算时干

### 十神

日干（日柱的天干）代表命主本人，以日干为中心与其他干支比较五行生克：

| 关系 | 同性 | 异性 |
|------|------|------|
| 克我 | 七杀 | 正官 |
| 我克 | 偏财 | 正财 |
| 生我 | 偏印 | 正印 |
| 我生 | 食神 | 伤官 |
| 同我 | 比肩 | 劫财 |

### 大运与流年

- **大运**：每 10 年一换的大运势周期，从月柱顺排或逆排
- **流年**：当年的干支对命局的影响
- **起运年龄**：根据出生日到下一个（或上一个）节气的天数计算

## Fortune 中的实现

Fortune **不自行排盘**（八字排盘算法极其复杂，需要精确的万年历数据），而是将排盘任务交给 AI：

```
用户输入：出生日期 + 时辰 + 性别
→ skill prompt 模板：{{birthDate}} {{birthTime}} {{gender}}
→ AI（DeepSeek/Qwen）自行排盘 + 解读
```

**Prompt 设计亮点**（`bazi.skill.ts`）：

- **格式约束明确**："年柱：甲（木）申（金）"——每柱一行、干支分行标注
- **禁止废话**："不要逐字解析含义（别展开'甲木代表什么性格'）"
- **焦点控制**："重点回到用户——围绕用户可能的当下处境展开聊"
- **时辰缺失处理**："一句话提醒时柱缺失会影响部分精度，然后直接跳过，不啰嗦"
- **去宿命化**："保持正向，趋吉避凶，不搞宿命论"

### 十二时辰选择

Fortune 提供完整的十二时辰下拉选择：
`子时(23-01) → 丑时(01-03) → ... → 亥时(21-23)`

## 在建构论框架中的位置

八字在 [[overview/占卜符号系统的可迁移性]] 中的分类：

| 维度 | 评估 |
|------|------|
| 符号可迁移性 | **极低**——符号来源于客观时间数据（干支历法），无法替换 |
| 解释规则层 | 固定（十神关系、大运流年推算规则） |
| 叙事建构空间 | 高——同样的八字不同的命理师有不同的解读重点 |
| 用户门槛 | 最低——只需出生日期，不需要学习任何符号 |

八字是最接近"科学化"的占卜形式（输入是客观的出生时间），但因为符号层与解释层完全解耦（同一八字不同命理师的解读可以大相径庭），它依然是一种符号建构——只是符号的"原材料"比塔罗牌更"硬"。
