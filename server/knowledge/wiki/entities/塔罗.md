# 塔罗

## 基本信息

- **类型**：纸牌占卜系统
- **构成**：78张（22大阿卡纳 + 56小阿卡纳）
- **起源**：15世纪意大利北部（米兰），最初为游戏用牌
- **占卜化**：18世纪起被系统化用于占卜
- **三大体系**：马赛塔罗、韦特塔罗、托特塔罗

## 结构与特征

### 双层架构

```
塔罗牌 78张
├── 大阿卡纳 22张（愚者→世界）
│   └── 功能：人生重大转折、灵性成长、命运关键转折
│   └── 影响：长期、深远、个人较难控制
│   └── 理论对应：荣格原型、卡巴拉22条路径
│
└── 小阿卡纳 56张
    ├── 权杖 14张（火）→ 行动、事业、激情
    ├── 圣杯 14张（水）→ 情感、关系、直觉
    ├── 宝剑 14张（风）→ 思维、冲突、决策
    └── 星币 14张（土）→ 物质、金钱、稳定
    └── 功能：日常生活事件、具体事务
    └── 影响：短期、暂时、个人可以控制
```

## 在占卜符号谱系中的定位

参见 [[overview/占卜符号系统的可迁移性]] 的框架：

| 维度 | 塔罗的定位 |
|------|-----------|
| **符号来源** | 漫长传统积累（非任意创造），有卡巴拉/占星作为"底层架构" |
| **可迁移性** | **中等**——牌义部分可替换，但四元素框架和大小牌分层是其核心结构 |
| **受众所需共识** | 需要认同塔罗符号学，门槛高于趣味占卜但低于八字 |
| **符号自洽性** | **极高**——78张牌的四元素+灵数学体系高度自洽 |

> 塔罗是"符号映射可迁移性"的**最佳参照系**：它证明了符号体系的自洽性（而非符号的神圣性）才是占卜效力的来源——因为塔罗的符号本身就是15世纪工匠的"人造设计"，却在数百年使用中被赋予了深厚的象征意义。
