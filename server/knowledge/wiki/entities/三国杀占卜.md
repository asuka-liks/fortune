## 基本信息

- **名称**：三国杀占卜（亦称"三国杀穿塔罗"）
- **来源**：抖音/B站短视频博主
- **核心创新**：用三国杀游戏卡牌替代传统塔罗牌
- **传播时间**：约 2025-2026 年

## 技法体系

### 抽牌规则
- 抽 2-3 张游戏牌 + 1 张武将牌
- 每张牌判断正位/逆位
- 牌面象征义 + 点数起卦双重解读

### 三层复合解读

```
┌─────────────────┐
│  第1层：卡牌象征  │  ← 牌名 + 游戏功能 → 生活隐喻
├─────────────────┤
│  第2层：卦象爻辞  │  ← 点数 ÷8÷6 → 本卦/互卦/变卦
├─────────────────┤
│  第3层：体用生克  │  ← 体卦/用卦/互卦 → 综合吉凶判断
└─────────────────┘
```

### 与塔罗牌的对比

| 维度 | 塔罗 | 三国杀占卜 |
|------|------|-----------|
| 牌数 | 78张 | 160+ 张（游戏牌+武将） |
| 符号体系 | 神秘学传统（数百年积累） | 游戏规则（即时共识） |
| 正逆位 | 有明确的牌义体系 | 按牌的功能判断（杀=主动，逆位=不该主动） |
| 随机化 | 洗牌 | 洗牌 / 系统随机发牌 |
| 起卦层 | 无（纯牌义解读） | **有**（点数 ÷8÷6 梅花易数） |
| 学习门槛 | 高（78张牌 × 正逆位 × 多维度含义） | 低（只要会三国杀规则即可参与） |

## 在 Fortune 中的实现

Fortune 项目的**趣味占卜模式**直接参考了这一流派：

- `server/utils/sanguosha.ts` — 三国杀卡牌定义和抽牌逻辑
- `server/utils/bagua.ts` — `deriveByPoints()` 数字起卦
- `server/api/fortune/fun-draw.post.ts` — 抽牌 + 起卦 + 格式化 API
- `config/skills/fun.skill.ts` — 趣味占卜的 system prompt

最终交付：AI 拿到"卡牌文本 + 卦象文本"后，模仿这一流派的解读风格生成回复。

## 文化意义

这一占卜流派是 **"共识性符号占卜"** 的生动案例：它证明了占卜的效力不来自某种古老权威的加持，而来自参与者对符号映射的**认同感**。三国杀玩家对"杀=主动""桃=回血"有高度共识——这套共识一旦建立，符号系统就能运作。

这也是为什么 Fortune 项目选择将它作为一个独立技能：**它比塔罗更轻量，比八字更通俗，比星座更接地气**。
