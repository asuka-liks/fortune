# 易经

## 基本信息

- **全称**：《周易》（周代易学）
- **构成**：《经》（六十四卦+卦爻辞）+《传》（十翼）
- **起源**：伏羲画八卦→文王演六十四卦→孔子作十翼（传统说法）
- **年代**：卦爻辞约公元前11世纪（西周初年）；易传约战国
- **地位**：群经之首，中国哲学的总源头

## 符号结构

### 二进制生成逻辑

```
阴爻（- -） + 阳爻（—）
  ↓ 2^3
八卦（三爻卦）: 乾兑离震巽坎艮坤
  ↓ 两两相叠 8×8
六十四卦（六爻卦）: 2^6 = 64种状态全覆盖
```

> 从数学角度看，六十四卦是一个2^6的**完全排列系统**——它不存在"为什么是78张"或"为什么是160张"的任意性问题。这是所有已知占卜体系中最"自洽"的符号结构。

### 三元架构

```
易经
├── 符号层：八卦→六十四卦（2^6全排列，无冗余）
│   └── 先天八卦（数序）：乾1兑2离3震4巽5坎6艮7坤8
│   └── 后天八卦（方位）：震东巽东南离南坤西南兑西乾西北坎北艮东北
│
├── 解释层：卦辞 + 爻辞（384条爻辞 + 64条卦辞）
│   └── 易传十翼（哲学化阐释）
│   └── 万物类象（梅花易数取象系统）
│   └── 六亲纳甲（六爻占卜逻辑）
│
└── 应用层：
    ├── 占卜（梅花易数/六爻/奇门遁甲）
    ├── 哲学（阴阳辩证/天人合一）
    ├── 中医（五运六气/经络学说）
    └── 风水（九宫飞星/玄空）
```

## 在占卜符号谱系中的定位

### 与塔罗的核心对比

| 维度 | 塔罗 | 易经 |
|------|------|------|
| **符号生成** | 工匠设计 + 600年积累 | 2^3 → 2^6 数学排布 |
| **符号自洽性** | 高（78张四元素体系） | **最高**（2^6全排列，无任意性） |
| **底层架构** | 卡巴拉（19世纪嫁接） | 河图洛书（内生） |
| **哲学深度** | 通过卡巴拉间接获得 | **直接内嵌**（《系辞》本身就是哲学经典） |
| **荣格关联** | 后世附会 | 荣格理论的**直接灵感来源** |
| **可迁移性** | 中等 | **极高**（万物类象 = 无限映射） |

### 易经作为"建构的原型"

从 [[overview/占卜符号系统的可迁移性]] 的建构论视角，易经的特殊地位在于：

> **它是所有符号占卜系统的原型——不是因为它是"神圣的"，而是因为它的符号建构逻辑是最透明、最数学化的。**

- 塔罗的78张为什么是78张？——没有严格数学理由
- 易经的64卦为什么是64卦？——因为 2^6 = 64，这是阴爻阳爻排列组合的必然结果

易经的符号体系不需要"神圣起源"来获取合法性——**它的自洽性来自数学**。
