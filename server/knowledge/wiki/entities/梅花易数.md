# 梅花易数

## 基本信息

- **创立者**：邵雍（邵康节，北宋）
- **得名**：赏梅见二雀争枝坠地，起卦预言次日有女子折梅跌伤，应验
- **核心原理**：万物皆数 + 万物类象
- **关键机制**：卦以八除，爻以六除 → 体用生克 → 应期推算

## 在符号可迁移性框架中的位置

梅花易数在 [[overview/占卜符号系统的可迁移性]] 的三层架构中占据特殊位置：

| 层 | 梅花易数的对应 |
|----|-------------|
| **第1层：符号映射层** | **万物类象**——八卦=天雷风火水山地泽=父长子中男少男母长女中女少女=马龙鸡雉鱼狗牛羊… |
| **第2层：解释规则层** | **体用生克**——体卦(自己)vs用卦(对方)，五关系(用生体/体用比和/体克用/体生用/用克体) |
| **第3层：叙事建构层** | **卦爻辞+直觉类比**——解卦者将符号结果编织为与问卦人情境相关的叙事 |

> 梅花易数是符号映射理论的**极限形态**：它不是在某个有限的符号集合中映射（塔罗78张、三国杀160+张），而是在**整个经验世界**中映射。任何事物——一个数字、一声鸟鸣、一件衣服的颜色——都可以通过"万物类象"映射到八卦。
>
> 这为 [[concepts/卡牌占卜符号映射]] 提供了方法论的天花板：如果连"今天的日期"都可以起卦，那任何有限符号系统的映射都是这个逻辑的特例。

## 核心算法

### 起卦公式

```
上卦 = 第一数 ÷ 8 取余 → 先天八卦（1乾2兑3离4震5巽6坎7艮0/8坤）
下卦 = 第二数 ÷ 8 取余 → 同上
动爻 = 第三数 ÷ 6 取余 → 1-5对应爻位，0=第6爻
本卦 = 上卦 + 下卦
互卦 = 本卦的2/3/4爻为下 + 3/4/5爻为上
变卦 = 动爻阴阳反转
```

### 体用生克矩阵

| 关系 | 判词 | 逻辑 |
|------|------|------|
| 用生体 | 大吉 | 环境/对方滋养你 |
| 体用比和 | 吉 | 双方和谐共振 |
| 体克用 | 小吉 | 你能掌控但费力 |
| 体生用 | 耗损 | 你在消耗自己 |
| 用克体 | 大凶 | 环境/对方压制你 |

## 与Fortune的接口

梅花易数的算法化程度极高——起卦过程完全是**确定性数学运算**：
1. 输入：任意数字（卡牌点数、时间戳、随机数种子）
2. 运算：÷8取卦 + ÷6取动爻 + 查表取万物类象
3. 输出：本卦/互卦/变卦 + 体用生克 + 应期

这正是 `server/utils/bagua.ts` 中 `deriveByPoints()` 的底层逻辑。参见 [[concepts/数字起卦与梅花易数]]。
