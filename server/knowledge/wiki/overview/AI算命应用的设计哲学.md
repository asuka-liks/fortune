## 一句话结论

**AI 算命的价值不在于"算得准"，而在于提供一种结构化的、有仪式感的、低心理门槛的自我审视方式。** Fortune 的设计哲学是：诚实比神秘感更重要，建构比通灵更诚实，建议比预测更有用。

## 核心命题

### 命题一：AI 不是通灵者，是叙事建构者

传统算命师通过知识、经验和话术来建构有说服力的叙事。AI 做的是一样的事——它有更广的知识面、更强的关联能力、更稳定的输出质量——但它**不是通灵**。

Fortune 在设计层面接受了这个事实，而不是试图掩盖它：

- 所有回复结尾的 "⚠️ 以上内容由 AI 生成"
- 趣味模式主动暴露建构过程（"X除以8余Y，对应Z卦"）
- 所有技能 prompt 中的 "不搞绝对化预测，强调自由意志"

### 命题二：占卜 = 有仪式感的自我审视

为什么人们觉得算命"准"？不是因为预测了未来，而是因为：

1. **符号触发了联想**：同一张塔罗牌，不同的人看到不同的东西（罗夏墨迹效应）
2. **仪式感创造了安全空间**：抽牌/摇卦的仪式感让用户允许自己谈论平时不会谈的事
3. **解读者充当了镜子**：好的解读不是在"告诉"用户什么，而是在帮用户发现自己已经知道但不敢面对的事
4. **叙事创造了意义**：人类通过叙事理解世界——占卜给混乱的生活事件提供了一个"故事线"

Fortune 的设计不是创造"更准的算命"，而是创造**更好的自我审视工具**——诚实、温暖、不装神弄鬼。

### 命题三：不需要"神圣"的符号

[[overview/占卜符号系统的可迁移性]] 已经论证了：任何自洽的符号系统 + 参与者共识 = 可运作的占卜工具。

Fortune 的五种技能恰好覆盖了符号"神圣程度"的整个光谱：

```
极"硬"（客观数据）                           极"软"（纯游戏）
八字 ← 星座 ← 塔罗 ← 八卦 ← 趣味
│         │       │      │       │
干支      黄道     78张    64卦    三国杀
历法      十二宫   神秘    符号    卡牌
（用户     （模糊  学传统   体系    （最诚实）
觉得     的宇宙               
最科学）  感应）
```

五种技能都能运作——不是因为符号有多神圣，而是因为每种技能都提供了：
- 一套**自洽的符号**（不论来源）
- 一套**可操作的解释规则**（不论复杂程度）
- 一个**会说人话的叙事者**（AI）

## 工程中的四层伦理设计

### 1. Prompt 层：非宿命化约束

每种技能的 prompt 都注入价值观约束——不是"应该这样写"，而是"绝对不能那样写"：
- 禁止："你注定会..."、"你们一定会分手"
- 鼓励："可以试着..."、"不妨考虑..."

### 2. UI 层：免责声明

每次回复末尾的 "⚠️ 以上内容由 AI 生成，仅供娱乐参考" 不是法律免责——是**持续提醒**：你在和 AI 聊天，不是和鬼神通话。

### 3. 系统层：免费额度限制

3 次免费对话 + IP 限流不只是成本控制——它阻止了用户对 AI 算命产生过度依赖。如果 AI 算命是"无限免费"的，用户会把它当成"随时随地可以问的免费心理咨询师"——这种关系对用户不安全。

### 4. 趣味模式：暴露建构过程

趣味模式是伦理设计的顶峰——它主动暴露符号建构的全部过程：

> "第一张牌点数11：11÷8 余3 → 3=离 → 上卦"
> "杀是主动，诸葛连弩是持续主动"

当用户看到"这玩意就是这样运作的"——神秘感消失，但趣味和价值还在。这证明了：**占卜不需要靠神秘感来维持价值**。

## 与同类产品的设计差异

| 设计维度 | 传统算命 AI | Fortune |
|----------|------------|---------|
| 人设 | "上知天文下知地理" | "像朋友聊天，不拽文" |
| 篇幅 | 长篇大论分章节 | 简短口语化，不分章节 |
| 预测方式 | "你今年会..."（绝对化） | "这个阶段可以试着..."（建议化） |
| AI 身份 | 隐藏（制造"大师感"） | 暴露（诚实声明AI生成） |
| 符号过程 | 黑盒（"卦象显示..."） | 白盒（趣味模式展示计算过程） |
| 学术腔 | 堆砌专业术语 | "别掉书袋" |

## 未决问题

- [ ] 当用户真的把 Fortune 当"算命"而非"娱乐"时，3次免费额度 + 免责声明够不够？
- [ ] 趣味模式的"强行附会"是否可能冒犯真正相信占卜的用户？
- [ ] 是否应该显式标注"这是一个 AI 建构的叙事，不是预测"——比免责声明更直白？
- [ ] 如果未来加入"严肃塔罗模式"（不搞笑、更沉浸），神秘感 vs 诚实性的平衡点在哪里？
